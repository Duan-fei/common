1/发送短信验证码
请求url：/smscode/
请求方法：GET
数据格式：string/json/xml
请求参数：
参数名称        是否必须        参数说明
mobile            是         手机号
image_code        是         图片验证码是用户输入的内容
image_code_id     是         图片验证码编号

返回成功：
errno='0'
errmsg='发送成功'
返回错误
errno=‘4301’
errmsg='发送失败'


2/注册用户
请求url：/users
请求方法：POST
数据格式：json
请求参数
参数名称        是否必须        参数说明
mobile           是            手机号
password         是            密码
sms_code         是            短信验证码

返回成功：
errno=‘0’
errmsg=‘注册成功’
data=user.to_dict()
注册失败：
errno='状态码'
errmsg：注册不成功


3/项目首页信息展示
请求url:/houses/index
请求方法:GET
数据格式:json
请求参数:无
返回结果:房屋基本信息(to_basic_dict())
房屋数据/图片信息/标题/price/



4/房屋详情页面
请求url:/houses<house_id>
请求方法:GET
数据格式:json

请求参数
参数名称        是否必须        参数说明
house_id         是           房屋id
user_id          是           用户id,判断用户身份

返回结果:
data = user_id,house=详情信息(to_full_dict())

errno=RET.ERR
errmsg='ERRMSG'


5/获取用户发布房屋信息
请求url:/user/houses
请求方法:GET
数据格式:json

请求参数:
参数名称        是否必须        参数说明
user_id          是          用户id,backref反向引用,

用户和房屋:一对多的关系  user = User()     user.houses

返回结果:
errno:0
errmsg:ok
data:house:houses_list


6/房屋列表页
请求URL:/houses
请求方法:GET
请求参数:

参数名称        是否必须        参数说明
area_id          否           房屋所属区域信息
start_date_str   否           用户选择的开始日期
end_date_str     否           用户选择的结束日期
sort_key         否           排序条件/默认按房屋最新发布的时间进行排序
page             否           页数/默认访问第一页

返回结果:
errno:RET.OK
errmsg:'OK'
data:houses=houses_list,total_page=总页数,current_page=page
